# XSS-site
A website for XSS hacking - devloped by @CrowdedFire1 for Sheffield Ethical Student Hackers.

## Set-up  
You MUST run this in the XSS-site directory! It will break anywhere else.  
```python -m venv venv```  
```pip install -r requirements.txt```  
```sh start.sh```  
Note: This installs composer as a global install  

## Flags
They are in the format FLAG{the flag value}
